This repository documents my journey of learning **Three.js** and exploring the world of 3D web development.

## Learning Progress

### 01 - Introduction to Three.js

This section covers the fundamental building blocks required to create a basic Three.js application:

- Scene
- Camera
- Object (Mesh)
- Renderer

These concepts form the foundation of every Three.js project and are essential for displaying 3D content in the browser.

## Goal

To build a strong understanding of Three.js fundamentals and gradually move towards creating interactive 3D web experiences and larger projects.
